## El Problema: "No encuentro nada"

Si un usuario escribe "zapatilla" y tu base de datos busca "Zapatillas", no encontrará nada. Los buscadores SQL tradicionales (`LIKE %query%`) son rígidos, lentos y frustrantes para el usuario moderno acostumbrado a Google.

## La Solución: Búsqueda Difusa (Fuzzy Search)

Implementamos algoritmos que entienden la intención del usuario, tolerando errores tipográficos y ortográficos.

### Capacidades

1.  **Tolerancia a Errores:** Si escribes "refigerador", el sistema entiende "refrigerador".
2.  **Ranking de Relevancia:** Los resultados exactos aparecen primero, seguidos de los parciales.
3.  **Filtrado Instantáneo:** Búsqueda en tiempo real (mientras escribes) sin recargar la página.

```javascript
// Ejemplo de configuración de Fuzzy Search
const searchOptions = {
  keys: ['nombre', 'descripcion', 'sku'],
  threshold: 0.3, // Nivel de tolerancia a errores (0.0 = exacto, 1.0 = cualquier cosa)
  distance: 100
};
```

### Resultado

Una experiencia de compra o navegación fluida donde el usuario siempre encuentra lo que busca, aumentando la conversión.
